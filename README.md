# wishMyFriend
Django Project for creating wishing friend.

# Installation

### Clone Repository

> git clone https://github.com/SwarupVishwas18/wishMyFriend.git

### Install Repositories

>  pip install -r requirements.txt

### Execute Project

> python manage.py runserver
